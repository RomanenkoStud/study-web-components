Study Web Components

Getting Started

Installation and Usage

  1. Clone the repository:
    2. git clone https://github.com/RomanenkoStud/study-web-components.git
  2. Navigate to the project directory:
    3. cd study-web-components
  3. Install dependencies:
    4. npm install
  4. To start the demo server:
    5. npm run dev -w demo
  5. Open your web browser and navigate to http://localhost:5173 to access the page.

Optional

To install the simple-timestamp library as a separate package, run: 

npm i @romanenko.pavlo/simple-timestamp

To install the numeric-input library as a separate package, run: 

npm i @romanenko.pavlo/numeric-input

To install the autocomplete-combobox library as a separate package, run: 

npm i @romanenko.pavlo/autocomplete-combobox
